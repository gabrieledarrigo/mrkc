# mrkc

`mrkc` generates text with the [Markov chain](https://en.wikipedia.org/wiki/Markov_chain) algorithm.  


## Usage

```sh
$ mrkc 100 < text.txt

Ulisse sul petto. Alto levossi de' riguardanti stupefatti il grido...
```
